eReuse.org Workbench

If you see this it means we couldn't connect to the eReuse.org Workbench server. Please, ensure that:

  • The correct OVA in Virtualbox is running and well-configured.
  • The phone is connected to the WiFi of the router you configured, not the general WiFi. Note that Android can warn you that this WiFi has no Internet. This is correct and normal -just dismiss the message.
  • Advanced: If Workbench is not in the default 192.168.2.2:5000, change the URL pressing the wrench above.
After this, press to the button of the wrench above and then OK to try again.
